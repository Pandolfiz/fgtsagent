# COMANDOS PARA CONECTAR AO GITHUB
# Execute estes comandos APÓS criar o repositório no GitHub

# 1. Remover remote atual (se existir)
git remote remove origin

# 2. Adicionar o novo repositório do GitHub
# SUBSTITUA "SEU_USUARIO" pelo seu username do GitHub
git remote add origin https://github.com/SEU_USUARIO/saas-fgts-react.git

# 3. Configurar branch principal
git branch -M main

# 4. Fazer push de todos os commits
git push -u origin main

# 5. Verificar se funcionou
git remote -v

# ✅ PRONTO! Seu projeto está no GitHub!
